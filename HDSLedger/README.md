# Always Remember Rito Theory

Maven Lifecycle has 8-stages

| Stage               | Command     |
|---------------------|-------------|
| Validation          | 😏          |
| Compilation         | mvn compile |
| Testing             | mvn test    |
| Packaging           | mvn package |
| Integration testing | 😏          |
| Verification        | 😏          |
| Installation        | 😏          |
| Deployment          | mvn deploy  |
